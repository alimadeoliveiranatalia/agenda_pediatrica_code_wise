## Início Rápido

#### 1. clone este repositório

```git
  git clone https://github.com/alimadeoliveiranatalia/agenda_pediatrica_code_wise
```
#### 2. acesse o diretório /front_end
```bash
  cd front_end
```
#### 3. abra o código no visual studio code

```bash
  code .
```
#### 4. Verifique se a extensão Live Server está instalada
1. [Extensão Live Server](https://github.com/ritwickdey/vscode-live-server-plus-plus)
